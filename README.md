# E-commerce Template (React + Vite + Redux Toolkit + Tailwind)

## Quick start

1. Extract the zip.
2. Install dependencies:
   ```
   npm install
   ```
3. Start dev server:
   ```
   npm run dev
   ```

## Notes
- Tailwind is already configured.
- Update `src/api/axiosClient.js` with your backend baseURL.
- This is a minimal starter: add components, pages, and styles as you need.
